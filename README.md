# MBD_audio

El objetivo de este proyecto es la clasificación de audios según su comportamiento. Se distinguen tres grupos diferentes: __sonido normal__ (correspondiente a un corazón sano), __sonido anormal__ (correspondiente a corazones con alguna patología o problema) y __artefacto__ (sonidos de ruidos aleatorios). Para ello se proponen dos tipos de modelos: LSTM y CNN. Para cada uno se ha llevado un procedimiento parecido. Sin embargo, cabe destacar que para el último modelo se ha llevado a cabo un proceso de extracción de características diferente al usado en el primer caso. Los resultados y todo el proceso de exploración y análisis de audios se encuentran en el archivo _audio_analysis.ipynb_; y las funciones usadas a lo largo del notebook se encuentran en el archivo _audio.py_.
